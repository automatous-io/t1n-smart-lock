# Enclosure

3D printable enclosure that houses the T1N Smart Lock PCB and drops
into the factory center console opening. Designed for the T1N
center console; fit verified on a 2005, other years 2002 to 2006 should match but are
unconfirmed.

Licensed under [CC BY 4.0](LICENSE.txt).

Files included:
- STL file for direct printing
- STEP file for modification in CAD software

See [ENCLOSURE.md](../docs/ENCLOSURE.md) for print settings,
material recommendations, and assembly notes.
